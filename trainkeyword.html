Train Custom Keywords

This page uses transfer learning on top of the pretrained 18-word speech-commands model. You only need ~15–25 recordings per keyword. Type each keyword name below, click Record, then say the word once within the 1-second recording window. Clicking Save Model downloads three files (<name>.json, <name>.weights.bin, <name>.metadata.json) to your Downloads folder — point keywordlistener.html at those three files to use the model.
